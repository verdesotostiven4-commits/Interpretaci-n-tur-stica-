# Rumicruz — Presentación interactiva profesional

Proyecto web listo para Vercel sobre el diagnóstico de la interpretación turística del patrimonio cultural en la comunidad de Rumicruz.

## Qué incluye

- 16 diapositivas interactivas con estructura académica.
- Diseño profesional con colores andinos, textura visual, tarjetas y transiciones.
- Visual SVG propio en `assets/rumicruz-visual.svg` basado en los elementos del documento: mapa, arqueología Puruhá, Centro de Tradiciones, Fiesta de San Pedro, fanesca y entrevistas locales.
- Notas de exposición para sonar natural y no leído.
- Mapa de navegación, barra de progreso y modo pantalla completa.

## Controles

- Avanzar: flecha derecha, espacio o botón →.
- Retroceder: flecha izquierda o botón ←.
- Mapa de slides: tecla `M` o botón **Mapa**.
- Notas de exposición: tecla `N` o botón **Notas**.
- Pantalla completa: botón **Pantalla**.

## Deploy en Vercel

Es un sitio estático. Solo importa el repositorio en Vercel y publica directamente; no requiere instalación ni build.
