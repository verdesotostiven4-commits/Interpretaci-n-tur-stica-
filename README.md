# Rumicruz — Presentación interactiva

Proyecto web para exponer el diagnóstico de la interpretación turística del patrimonio cultural en la comunidad de Rumicruz.

## Uso

- Avanzar: flecha derecha, espacio o botón →.
- Retroceder: flecha izquierda o botón ←.
- Mapa de slides: tecla M o botón Mapa.
- Notas de exposición: tecla N o botón Notas.
- Pantalla completa: botón Pantalla.

## Vercel

Es un sitio estático con `index.html`. En Vercel puedes importarlo desde GitHub y publicarlo directamente.

La presentación está pensada para verse profesional, creativa y con movimiento, pero con estilo de exposición estudiantil natural.
